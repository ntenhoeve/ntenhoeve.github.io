Introspect Framework Concepts

An Introspect Application class basically provides all the information needed (configuration) to start the Introspect Framework.

The Introspect Framework favours configuration by code over configuration by configuration files (see “Code or configuration files” chapter in Martin Fowlers article for the arguments why). Configuring the Introspect Framework configuration is done by implementing the IntrospectApplication interface. If you favour to use configuration files, it should not be difficult to write an application class that implement the IntrospectApplication interface which gets the information from a configuration file.
